# Portfolio

* 순수 HTML, CSS, JavaScript로 제작한 개인 포트폴리오 사이트 
* React 학습 전 기본 개념 DOM 조작, 이벤트 처리, 비동기 통신, 상태 기반 렌더링 흐름을 학습하기 위한 프로젝트

## 배포 URL

- GitHub Pages: `https://7eerup.github.io/portfolio/`

## GitHub 저장소

- Repository: `https://github.com/7eerup/portfolio`

## 사용 기술

- HTML5
- CSS3
- JavaScript
- GitHub API
- GitHub Pages

## 주요 기능

- 시맨틱 마크업 기반 레이아웃
- 반응형 네비게이션
- 모바일 햄버거 메뉴
- 다크 모드
- localStorage를 활용한 테마 저장
- 스크롤 탑 버튼
- 스크롤 위치에 따른 네비게이션 스타일 변경
- Intersection Observer 기반 스크롤 애니메이션
- GitHub API 저장소 목록 조회
- 로딩 / 성공 / 에러 / 빈 상태 UI
- 프로젝트 필터링
- 문의 폼 유효성 검사
